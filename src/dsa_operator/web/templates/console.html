How an observing plan works

1. Stage a plan (below, or just ask the assistant) — it does nothing yet.   2. Preview bring-up to see the exact steps.   3. Arm it. Once armed, the autopilot runs the bring-up automatically (point → fringe-stop table → modes → start/restart fleet → wait for warm-up → utc_start), one step every ~30 s, for as long as you hold the executor lease. Use Advance bring-up to step it by hand / see where it is.
